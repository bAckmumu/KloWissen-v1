KloWissen-v1
============

KloWissen is a Raspberry Pi podcast player. This is the First version of it.

## Linux applications
* Music Player Daemon 
* gPodder

## cron Jobs
* crontab -e
* 01 * * * * /home/pi/script/loadpodcasts
* 05 * * * * /home/pi/script/deleteday >> /home/pi/script/del.log

## Phython Skripte
* deletday
* loadpodcast
* player
